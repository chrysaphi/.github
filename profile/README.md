# Bridging Gaps in Heliospheric Radio Data Analyses

A Lorentz Center workshop bringing together radio heliophysicists and
scientific software developers to address challenges in heliospheric
radio data accessibility, standardisation, and usability.

- Lorentz Center, Leiden, NL
- 18–22 May 2026

## About the workshop

Despite the wealth of heliospheric radio data now available from
ground- and space-based instruments, broader scientific use remains
limited due to challenges in accessing datasets and the lack of
standardised tools to read and process the diverse data formats.

This workshop aims to:

- Catalogue and consolidate heliospheric radio data sources
- Survey and compare existing analysis tools
- Develop a standardised data container within the SunPy ecosystem
- Produce a community-driven roadmap for the field
